# sendline 与 send

sendline 比 send 在数据末尾加一个换行符，在 read 进行输入操作的时候要格外主要 send 和 sendline 的使用时机。
当 read 读取 IO 的字节数量为 40 个时，我们用 sendline 输入了 40 个字符串，但是末尾残留一个换行符，可能会导致下一次 read 直接读取这个缓冲中的换行符导致输入数据错误
