# 当有负数参与时

首先进行与正数相同的位运算 (负数都是补码表示的)

# & | ^

# <<>>

对于 << >> 移位操作
<< 二进制整体左移 (符号位和前面的去掉)，会在低位补 0, 不论正负都一样
>> 二进制整体右移

  • 对于正数，会在高位补 0
  • 对于负数，会在高位补 1
